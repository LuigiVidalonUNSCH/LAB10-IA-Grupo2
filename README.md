### Descripción breve:
Este proyecto implementa un prototipo de IA para e-commerce.
El Chatbot (implementado con OpenAI y Dialogflow) utiliza un catálogo local para responder preguntas sobre precio y stock de productos.
El Recomendador utiliza un algoritmo de filtrado colaborativo simple (basado en historial de compras de prueba) para sugerir nuevos productos a los usuarios, basándose en lo que el cliente ha comprado y recomendándole otros productos relacionados.
